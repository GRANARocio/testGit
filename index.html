GIT

Partie 1 : configuration de GIT

Création d'un dépôt local → utilisation de "git init"
Commande à utiliser → git init

Vérifier que GIT est bien lié à votre compte GITHUB → utilisation de "git config"
Vérification du nom d'utilisateur → "git config --global user.name"
Vérification de l'adresse email → "git config --global user.email"
Si la liaison de votre git à votre github n'existe pas alors faites ceci :
Pour le nom d'utilisateur → "git config --global user.name suivi de votre nom d'utilisateur sur github"
pour l'adresse email → "git config --global user.email suivi de l'adresse mail utilisée sur github"
